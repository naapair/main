POS Lot Selection and Restrict Negative Stock Product

    Key Features
  • Using this module you can select Serial Number of Productin POS interface.
  • Using this module you can select Lot Number of Productin POS interface.
  • You can remove product when Lot/Serial Number Expired.
  • Restrict Negative stock Lot/Serial Product.
  • You can add large number of Product with Lot/ Serial Number.
  • Easily search product with Lot/Serial Number.
  • Auto Disable product once its not is Stock.

POS Serial Selection (with Restrict Negative stock Product)
Create a Product with Tracking by Serial Number.

View of Created Product with Serial number.

Configuration for display only realtime available product quantity in POS Interface.

POS Serial Number Configuration and configuration for a Release days(Product Serial Expiry days).(This is product sometime this product we need to user for bakery product than we need to mentined auto removal time as product expiry date.)

Create order and select Serial number of product. Here You can see list of available Serial Number of product.

If you will add serial number with more that available product quantity than it will restirct you, and only allowed avaialble product serial number.

View of added Serial number in Cart. Here You can search product with serial number if you have large number of the serial number available.

Create Order and proceed for Payment. And here Select Auto invoice True for get serial number in Invoice.

Selected Serial number will Print in the POS Receipt.

View of Serial Number in POS Order.

View Serial Number in Stock Picking.
