Create Product From POS

Create products directly from POS on the fly and keep the customers queue moving. The module lets you create and add the product to the cart directly from the POS session.

Features:

  • A seller can create from Odoo POS at any time in POS Session.
  • A seller can create new product in offline mode also
  • No Backend configuration, just install and use.

By Default, You Cannot Create Product From POS In Odoo!!

Inherently, there is no option to create a product from POS during a running session. The products can only be created in Odoo backend and selected while running the POS session. To add a new product to the POS, the product needs to be created at the backend and then the POS session reloaded.

What Does "Create Product From POS" Does?

As the name suggests, Odoo Create Product From POS module allows you to create a product from Odoo POS directly during a running session. The product can be instantly created in Odoo POS and used in the cart in a running session & also saved at the Odoo backend.

Problems Fixed Through The Module:

No settings to Create product directly in Odoo POS running session

There is no feature available in default POS operational settings to create a product directly in Odoo POS running session. The POS user first needs to create the product in the Odoo backend then reload the POS session. Only then, the product becomes available in the Odoo POS. The module cut-shorts and lets you create the product in the POS session itself and it to the cart simultaneously

No Product exists in the Odoo POS for a scanned barcode

In case the POS returns no result for a scanned barcode of a product, the POS user can easily create the corresponding product associated with the barcode instantaneously from the running POS session. No need close the POS and go to the Odoo backend.

What happens If the POS is running In Offline Mode?

The module works even when the POS is running offline. The seller can create the product and add it to the cart in online as well as in offline mode and validate the order with the newly created product. However, the product would only be synced and saved in the Odoo backend when the POS comes online.
