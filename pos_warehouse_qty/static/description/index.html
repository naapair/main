Odoo 14 POS Warehouse Quantity (Community)

This module displays product stock in POS

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • Enable configuration of Display Stock for use the feature.




  • Select the Product from the cart and click on the warehouse icon.


  • Display Stock of Product with Warehouse and it's locations and print selected Product stock Details on clicking print icon.


  • Print Product stock Details on clicking print icon.
